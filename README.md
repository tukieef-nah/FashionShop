## Project Scope

Phạm vi của dự án

- Đăng nhập/Đăng ký/Đăng xuất.
- Quản lý - Admin:
  - Quản lý sản phẩm
  - Quản lý hóa đơn.
  - Quản lý tài khoản.
  - Quản lý danh mục.
  - Quản lý thương hiệu.
- Hoạt động chính:
  - Xem sản phẩm
  - Thêm sản phẩm vào giỏ hàng.
  - Đặt hàng.
  - Thanh toán.
- Cá Nhân: Quản lý thông tin cá nhân.
